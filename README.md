# sistema-rag-tfg
Presento mi arquitectura RAG (_Retrieval Augmented Generation_) para TFG o cualquier otro proyecto:
  - Notion como base de datos de entrada (PDFs, páginas web, vídeos de Youtube, imágenes)
  - Extracción multimodal (LlamaParse/trafilatura/Whisper/Gemini)
  - Almacenamiento vectorial en ChromaDB
  - Consulta vía chat UI
  - Scripts de administración para deduplicación, borrado de recursos y actualización de flags en Notion.
